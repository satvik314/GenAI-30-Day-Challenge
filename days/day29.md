### Part 1: Understanding Perplexity AI (2 min)

**What is Perplexity AI?**

Perplexity is an AI-powered answer engine that:
- Combines language models with real-time web search
- Provides cited sources for all information
- Specializes in research and learning
- Always has access to current information
- Formats answers with references
- Offers different research modes

**Key Difference from ChatGPT**:

```
ChatGPT:
- Trained on data up to a cutoff date
- No real-time information
- No citations by default
- Great for general chat

Perplexity:
- Always searches the web
- Current information guaranteed
- Every claim cited with links
- Designed for research
```

**When to Use Perplexity**:

- Researching current events
- Finding latest scientific findings
- Learning about recent developments
- Academic research
- Fact-checking with sources
- Industry trends and updates
